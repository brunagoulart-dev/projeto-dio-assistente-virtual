# FinAssist AI

FinAssist AI é um assistente virtual inteligente para relacionamento financeiro, desenvolvido em Python com Flask, com foco em experiência do usuário, linguagem natural, respostas contextualizadas e simulações financeiras simples.

## Objetivo do projeto

O projeto foi criado para atender ao desafio de desenvolvimento de uma experiência digital voltada ao relacionamento financeiro, guiada por inteligência artificial generativa e fundamentada em boas práticas de UX.

A solução busca oferecer interações claras, seguras e personalizadas, permitindo ao usuário:

- tirar dúvidas sobre produtos financeiros;
- receber explicações em linguagem simples;
- realizar simulações demonstrativas de empréstimo;
- realizar simulações demonstrativas de investimento;
- manter contexto básico ao longo da conversa.

## Funcionalidades

- FAQ inteligente com base em palavras-chave
- Simulação de empréstimo com juros compostos
- Simulação de investimento com capitalização mensal
- Continuidade de contexto em perguntas subsequentes
- Interface web amigável
- Reinício de conversa

## Tecnologias utilizadas

- Python 3
- Flask
- HTML5
- CSS3
- JavaScript
- JSON
